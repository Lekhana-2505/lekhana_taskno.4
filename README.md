# 🔐 Login Authentication System

A simple login authentication system built using Python that allows users to register, login, and access a secured page.

---

## 🚀 Features

- User Registration (Sign Up)
- User Login (Sign In)
- Password validation
- Secure page access after login
- Simple file-based storage system

---

## 🛠️ Technologies Used

- Python 3
